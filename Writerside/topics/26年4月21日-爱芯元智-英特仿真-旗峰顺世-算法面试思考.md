# 26年4月21日_爱芯元智+英特仿真+旗峰顺世_算法面试思考
<br/>
&emsp;&emsp;春招期间，虽然我对岗位和薪资的要求变高了，但任然可以感觉算法岗的面试邀约变多了。经过这三个公司的算法面试，
总结了算法岗需要的一些技能和面试技巧以及自己所缺少的能力。
<br/>
&emsp;&emsp;在爱芯元智的面试中，面试官着重考察了我C++关于STL和智能指针相关的知识点，并问了我关于算子是什么，在模型训练和推理中的作用，
我没有回答上来，这个岗位应该是一个ai infra岗，设计一些写算子的工作。
<br/>
&emsp;&emsp;在英特仿真的面试中，面试官比起考察我公司相关的业务，更希望我能展示一下我这三年来，使用了什么算法解决了什么问题，并基于此进行提问，
但由于我事前没有做好准确，且缺少总结的能力，所以回答的比较差。
<br/>
&emsp;&emsp;在旗峰顺世的面试中，首先进行了一场笔试，考察了大模型的各个方面，主要为当训练出现问题是，应该基于现状如何排查问题，也考察了手写GPT2，
，还有贝叶斯算法等。面试中，面试官主要问了我如何评判模型的好坏，如何设计目标函数，设计prompt的策略等等，还有LoRA的秩设定的原因，
也问了我过没过六级和C++用过没有。
<br/>
&emsp;&emsp;综合来讲，C++是所有算法岗的必备知识，同样的在大模型的训练和推理中，也要更加关注底层原理，模型效果的好坏也应该要能找到原因。
后续应该补足下面知识点：
<br/>
&emsp;&emsp;1.贝叶斯算法；
&emsp;&emsp;2.LoRA微调各个参数设定的逻辑，秩的设定；
&emsp;&emsp;3.对于prompt设计，更专业的说法；
&emsp;&emsp;4.模型从pt转到onnx中，底层有哪些变化；
&emsp;&emsp;5.在心理咨询机器人项目中LoRA微调的损失值是如何变化的；
&emsp;&emsp;6.如何设置目标函数；
