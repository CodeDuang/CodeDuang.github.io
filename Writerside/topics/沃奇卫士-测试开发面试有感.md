# 沃奇卫士_测试开发面试有感

在25年11月17日下午2点，参与了美国外企watchguard的测试开发实习生面试，时长共3h，收获颇多

#### 公司介绍
<br/>
&emsp;&emsp;WatchGuard（沃奇卫士）是美国网络安全解决方案供应商，
成立于1996年，总部位于华盛顿州西雅图。公司专注于防火墙、VPN及统一威胁管理（UTM）领域，
核心产品包括Firebox系列硬件设备和Fireware操作系统。（来自百度）
<br/>
&emsp;&emsp;公司位于成都银泰中心，类似于万物纵横的办公室，在中国为全球测试中心（唯一），2016年北京的开发中心被解散，
在世界各地的其它国家也都有不同的开发中心，保证双休且同事关系简单，最近有新的ceo加入公司，来自竞品公司，
老ceo也没有离开，而是加入了董事会，薪资9k-12k,平均每年会有一次调薪机会,每个季度不同项目组会自己组织聚餐，
年终会有年会，公司并非大厂，但也算小而美的企业，且美国外企注重人权，
每个月会有500元出行餐补助，直接加入工资。基于推测，似乎公司不存在出差，但定期会有外国同事过来，
与外国同事交流开会的方式为线上会议。（还没问每天的早晚上班时间和午休，以及年终奖）

#### 笔试回忆
<br/>
题目组成：涵盖了：Docker知识、浏览器通信原理、python处理字符串匹配问题、python处理大量数据如何优化、部分简单的英语问题等

问题1：用Docker启动一个Django应用，并持续运行，同时对于局域网开放某个端口访问，请分步骤描述应该如何做，并且每个步骤的命令是什么？

问题2：当一个用户打开了浏览器，输入某个网址，按下回车，请问客户端、服务端、网络分别会有什么样的变化，底层会发生什么样的信息传递？

问题3：给你一个字符串，请将字符串中“ab”、“aab”、“aaab”...("b"前面有n个连续的"a"，n为任意正整数)的"a"全部删除，达到让"b"前面没有"a",
然后输出处理后的字符串。

问题4：请写一个函数，针对于银行卡号进行判断，根据首k个数字(k为2-4)，判断属于哪个银行，根据每家银行规定的长度信息，判断是否准确，
无法确定银行的输出Unkonw。


#### 面试回忆


我很赞成面试官的理论，任何岗位的面试，我们应该注重的是基础、学习、思考等能力，
而不是问你什么什么工具用过没怎么用。

面试官不是很赞赏我对于问题的处理（不关注背景故事，而是只关注输入输出要求），
或许在实际的项目中，搞懂业务逻辑同样重要。

对于倒数第二道题，我用栈的思想，一个一个字符入栈，如果遇到b，则循环出栈栈顶a，直到栈顶不是a，然后入栈b。
面试官让我观察整体，在python中有更好的解法。

对于最后一道题，输入一个银行卡号，我们需要判断这个银行卡号是哪个银行的，同时判断长度是否符合要求，
由于python中没有switch函数，我使用了多个并列的if-elif-else函数来进行筛选，
这个做法应该有更好的解决方法，基于python的函数。
