# 枚举类与枚举成员

python的枚举用法一般用于给程序中的某些常量赋予可读性。

```python
#导入枚举库
from enum import Enum

class RUN(Enum):
    left: float = 1.5
    right: str = "right:2.5"
    forward: bool = True
    backward: int = 3
    

#调用枚举类
print(RUN.right.value) # right:2.5
print( type(RUN.left),type(RUN.right),type(RUN.forward),type(RUN.backward) ) # <enum 'RUN'> <enum 'RUN'> <enum 'RUN'> <enum 'RUN'>
print(RUN["backward"].value) # 3
```
