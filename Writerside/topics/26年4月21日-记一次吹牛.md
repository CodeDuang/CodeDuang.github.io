# 26年4月21日_记一次吹牛
<br/>
&emsp;&emsp;我在很多人的场景下，被某个人夸了一句，然后大家都很愿意听我说我更好的经历，所以我就顺着众人和我的虚荣心继续往下面编故事。
事后回想起来，发现这么做对自己毫无好处，也不明白自己当时为什么会变得不像自己，像是虚荣心让我觉得现实生活配不上自己，
我应该如故事中一样。
<br/>
&emsp;&emsp;我想，这本质上是自卑的一种表现，使我不愿意接纳我的过去和现状，我喜欢别人羡慕的目光，并引以为傲，
并且工作中周围同事都是高学历且能力很强的，让我有被现充伤到的挫败感，所以需要在其他人身上找优越感，欺软怕硬。
对于这种情况，我需要学会接纳自己，荣辱不惊，这是我一直想要做到的，但似乎我总是要先到达山峰，才能对山腰上的人宠辱不惊，
而对更高山峰的人感到自卑，这是我应该攻克的课题。
<br/>
&emsp;&emsp;在编故事方面，我也发现我抓不到重点，总是语义混乱，表达不清，每段话的信息密度很低。也有很多口头禅，
说不定还有地方口音。日常生活中应该慢慢锻炼，使得自己普通话更好，表达更有逻辑！
